Hash trails

Maps of the trails run by Hash House Harrier packs in London

Dec 4, 2020

Several Hash House Harrier groups run all over London each week. Each week the location of the trails varies as that week’s “hare” lays the trail in an area they know well. I have been collecting the GPS routes of the trails from Strava for each group and plotting them on a map using Python.

If you are interested in running/drinking with one of the Hashes, check our when and where upcoming hash trails will be on the London Hash House Harriers site or the London Hashing Site (LaSH).

Most recent trails

The map below shows the trails run over the last week (or so) by the various Hash House Harrier groups around London. You can zoom in on the map, and click on the location icon to get details of which group each trail belongs to.
