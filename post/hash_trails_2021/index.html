Hash House Harriers trails 2021

Map of the trails run by various Hash House Harrier across London in 2021

Apr 19, 2021

The London Hash House Harriers (LH3) run weekly all over London. Using GPS data from Strava, I have created a map (below) showing the routes run so far this year.

Due to Covid-19 lockdown restrictions, we were only able to begin running again in April, and even these runs were socially distanced (no running the trail in a big group, no pub at the end). As restrictions are eased we hope to get back to the traditional hashing format soon.
